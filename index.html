▮ The forge대장간의 도구

⚡ Local judging (C++ · Python)⚡ 로컬 채점 (C++ · Python)

Open a problem and its samples are extracted and judged instantly. LeetCode gets both AI-harness local judging and remote runs on the real LC judge.

문제를 열면 예제가 자동 추출되고 즉시 채점됩니다. LeetCode는 AI 하니스 로컬 채점 + 실제 LC 저지 원격 채점을 모두 지원.

💡 Staged AI hints💡 단계별 AI 힌트

Observation → approach → algorithm → full solution. Hints unlock only after 3 failed attempts — you build muscle, not answer-peeking habits.

관찰 → 접근 → 알고리즘 → 전체 풀이 4단계. 3회 이상 틀려야 열립니다 — 답을 보는 게 아니라 스스로 부딪히는 훈련.

📝 Auto review notes📝 오답노트 자동 발행

On solve, a review note is generated from your actual failure log — including a locally-verified model solution — and published to your blog.

풀이 완료 시 실패 기록을 근거로 오답노트 생성 — 로컬 채점으로 검증한 모범 답안 포함. 블로그로 발행됩니다.

🔮 Pattern recognition drills🔮 유형 인식 훈련

Guess the algorithm type before opening hints, then compare with official tags. Types you keep missing become weakness data with one-click drills.

힌트 전에 알고리즘 유형을 예측하고 공식 태그와 대조 — "못 알아본 유형"이 약점 데이터로 쌓이고 원클릭 드릴로 이어집니다.

🔁 Spaced repetition🔁 간격 반복 복습

Re-solve from scratch at 1/7/30/90-day intervals. Flashcards, review queues and retry queues bring problems back right before you forget them.

1/7/30/90일 주기 백지 재풀이. 플래시카드·복습 큐·재도전 큐가 잊어버릴 타이밍에 문제를 다시 가져옵니다.

🏁 Contest mode🏁 실전 모드

Virtual contests with performance estimates, speedruns, upsolving import, weekly coach reports and streaks — the habit loop that raises rating.

가상 대회 + 퍼포먼스 추정, 스피드런, 업솔빙, 주간 코치 리포트, 스트릭 — 레이팅을 올리는 습관 고리.

▮ Forge rules대장간의 규칙

  • Struggle first. Hints unlock only after 3 failures — building muscle beats reading answers.
  • Records are skill. Every attempt, failure and review becomes data that exposes your weaknesses.
  • Submit by hand. No automated submissions — we respect Codeforces policy.
  • Your data stays yours. Study records and API keys live only on your machine.
  • 스스로 먼저. 힌트는 3회 실패 후에만 — 정답 열람이 아니라 근육을 만드는 순서로.
  • 기록이 실력. 모든 시도·실패·복습이 데이터로 남아 약점을 드러냅니다.
  • 제출은 손으로. 자동 제출은 하지 않습니다 — Codeforces 정책을 존중합니다.
  • 내 데이터는 내 컴퓨터에. 학습 기록과 API 키는 로컬에만 저장됩니다.

▮ Install설치 방법

  1. Download the DMG above and open it
  2. Drag Solvesmith into Applications and launch it — that's it. The app is signed & notarized by Apple, so there are no security warnings.
  3. On first launch it self-diagnoses your environment and tells you how to install anything missing
  1. 위 버튼으로 DMG를 받아 엽니다
  2. Solvesmith를 응용 프로그램에 드래그하고 실행 — 끝입니다. Apple 서명·공증이 완료되어 보안 경고가 없습니다.
  3. 첫 실행 시 환경을 자가 진단하고 부족한 도구의 설치법을 알려줍니다

⚙️ Prefer the command line? curl -fsSL https://gwonsoolee.github.io/solvesmith-releases/install.sh | bash installs and launches in one line. Updates are automatic from v1.9.1 — the app updates itself in the background. ⚙️ 터미널을 선호하면 curl -fsSL https://gwonsoolee.github.io/solvesmith-releases/install.sh | bash 한 줄로 설치됩니다. v1.9.1부터는 자동 업데이트 — 앱이 백그라운드에서 스스로 최신 버전을 받습니다.

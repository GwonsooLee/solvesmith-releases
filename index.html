Codeforce Helper

Codeforces · LeetCode · AtCoder 훈련을 한 앱에서.
로컬 채점 → 단계별 AI 힌트 → 오답노트 자동 발행 → 간격 반복 복습까지, 알고리즘 실력 향상 루프 전체를 돌립니다.

⬇ macOS용 다운로드 Apple Silicon (M1 이상) · 무료

주요 기능

⚡ 로컬 채점

문제를 열면 예제가 자동 추출되고 C++/Python 코드를 즉시 채점합니다. LeetCode는 AI 하니스 로컬 채점 + 실제 LC 저지 원격 채점을 모두 지원.

💡 단계별 AI 힌트

관찰 → 접근 → 알고리즘 → 전체 풀이 4단계. 3회 이상 틀려야 열리는 스파르타 모드로 스스로 부딪히는 훈련을 강제합니다.

📝 오답노트 자동 발행

풀이 완료 시 실패 기록·힌트 사용을 근거로 오답노트를 생성 — 모범 답안 포함, 로컬 채점으로 검증된 코드. VitePress 블로그로 발행됩니다.

🔮 알고리즘 예측 훈련

힌트를 열기 전에 문제 유형을 예측하고 공식 태그와 대조 — 적중률과 "못 알아본 유형"이 약점 데이터로 쌓입니다.

🔁 간격 반복 복습

1/7/30/90일 주기로 백지 재풀이. 플래시카드·복습 큐·재도전 큐가 잊어버릴 타이밍에 문제를 다시 가져옵니다.

🏁 실전 훈련

가상 대회 + 퍼포먼스 추정, 스피드런, 업솔빙 가져오기, 주간 코치 리포트, 스트릭 — 레이팅을 올리는 습관 고리.

설치 방법

  1. 위 버튼으로 DMG를 받아 열고, 앱을 응용 프로그램 폴더로 드래그
  2. 최초 1회 — 터미널을 열고 아래 명령 실행 (Spotlight에서 "터미널" 검색): 
    xattr -cr "/Applications/Codeforce Helper.app"
  3. 앱 실행 → 시작 시 환경을 자가 진단하고 부족한 도구의 설치법을 알려줍니다
"손상되었기 때문에 열 수 없습니다"라고 뜨나요?
앱이 손상된 것이 아닙니다 — 서명되지 않은 앱을 인터넷에서 받으면 macOS가 표시하는 보안 안내입니다. 위 2번의 터미널 명령 한 줄이면 해결되며, 이후에는 계속 정상 실행됩니다.

정책

Codeforces는 자동 제출(봇)을 금지합니다. 이 앱은 제출을 자동화하지 않으며 코드 복사로 수동 제출을 돕기만 합니다. AI 힌트는 학습 도구로 설계되었습니다 (가상 대회 중 잠금, 스포일러 단계 제한). 모든 학습 기록과 API 키는 사용자의 컴퓨터에만 저장됩니다.
